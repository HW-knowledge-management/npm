# npm

# 환경설정
1. 배포 완료하였으므로 pull로 파일 받아오기
2. https://www.npmjs.com/package/hw-knowledge-management
3. npm 파일 설치하기
```sh
$ npm i hw-knowledge-management
$ npm install -g ./

# 설치 확인
$ npm list -g
```
4. 실행
```sh
$ node index.js
```
5. 수정 후 배포
```sh
$ npm publish
```

# 주의사항
- 팀원 대화 후 배포 필요
- pullRequest시 Branch 새로 만들어서 올리기
- pull 작업 후 push 하기
